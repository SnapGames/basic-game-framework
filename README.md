## Context

This Basic Game Framework on the Java platform is all you need to start coding a game on java with the JDK only.

## Compilation

Simply mavenized project with a :

```bash
 $> mvn clean install
```

## Execution

Run the app with :

```bash
 $> mvn exec:java
```
